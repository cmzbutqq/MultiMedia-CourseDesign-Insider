# 拓展功能（本阶段不纳入「重构方案」范围）

在 Web 版与现 C++/OpenGL 版**功能对齐**之后，可按下述方向扩展。**视觉与互动优先**，不追求严格广义相对论数值解。

## 已列方向

1. **更多天体类型**
   白洞、虫洞、中子星等——在光线步进或背景场公式中增加可切换的“度规/折射场”预设，配独立材质与调色。

2. **多天体与简易轨道**
   多天体场景、飞船/相机轨道、天体间引力——可先 **伪 N 体或 Kepler 近似** + 粒子/轨迹线；大规模再考虑 Worker 或 GPU 假 compute。

3. **摄像头手势交互**
   用 **MediaPipe Hands**（或同类浏览器方案）将捏合/拖动/旋转映射到「选中对象、平移、旋转」或 UI 焦点，与鼠标并存。

4. **进阶视觉物理感** 
   黑洞 **角动量（Kerr 观感近似）**、**相对论性多普勒 / 束宽**——在现有色球与吸积盘项上乘以可调的相对论因子，强调观感而非论文级精度。

5. **抗锯齿**
   针对步进噪声与硬边的 **超采样 / TAA / FXAA / MSAA（视 WebGL 能力）** 或与后处理结合的边沿平滑，在性能预算内折中画质与帧率。

6. **模拟时间扭曲**
   用可调的 **引力时间膨胀近似**（如随势阱深度的局部时钟缩放）驱动粒子、吸积盘动画、轨迹演化的「局部快慢」，可与后续 Web Audio 调制衔接；强调戏剧化观感而非严格史瓦西度规计时。

## 后续可追加（备忘）

- 时间缩放、脚本镜头（Keyframe）、录制/分享参数 JSON。
- Web Audio：靠近强引力区/高速运动时的氛围调制。
- XR（WebXR）仅观看模式：立体景深 + 头动，仍可与同一渲染目标对接。

技术选型与分期见重构上线后的单独设计；**当前以 `重构方案.md` 为唯一交付范围。**
